Failed to start component [StandardEngine[Tomcat].StandardHost[localhost]

构建了一个maven项目，tomcat发布的时候报错：

触发条件

​ maven项目中引入了这个依赖。

错误原因

​ mingsoft的shiro-freemarker-tags插件引入了javax.servlet-api包，和项目中其他依赖引入的servlet-api冲突(其实和tomcat自带的servlet-api也冲突)

解决方案

​ 最简单方便的方法。单独去掉javax.servlet-api依赖。

方案验证

​ Tomcat正常启动
